# 🎮 Gaming Platform Unity Integration Package

Integrate your Unity game with the **Gaming Platform Server** in minutes.

This Unity package allows your game to:

- Connect to the Gaming Platform backend
- Register as a playable server-listed game
- Submit player scores securely
- Participate in platform-wide competitions and leaderboards

---

## 🚀 Features

- Simple Unity integration
- Secure score submission
- Game registration via unique identifier
- Server-side validation
- Lightweight and easy to use

---

# 📦 Installation

### 1️⃣ Import the Unity Package

1. Download the `.unitypackage`
2. In Unity go to:  
   `Assets → Import Package → Custom Package`
3. Select the file
4. Import all contents

---

# 🔑 Setup

To integrate your game, you need:

1. **Game Identifier**
2. **Score Submission Secret**

These are provided by the platform administrator.

> Contact the platform administrator to receive your credentials.
